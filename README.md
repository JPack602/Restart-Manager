# Windows Restart Manager のサンプルコード
Restart Managerを使ったサンプルコードです。  
  
Restart Managerは本来Windowsの起動時や再起動時にアプリケーションを終了させるための機能です。  
使用例として、あるアプリケーションにより開かれているファイルを強制的に削除したい場合などに使用します。
